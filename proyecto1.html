Detalles Técnicos

Seguridad

  • Bastion Host para acceso SSH seguro
  • Grupos de seguridad configurados

Tecnologías

  • EC2 para servidor web
  • RDS PostgreSQL
  • Terraform para IaC

Este proyecto implementa una arquitectura clásica en AWS con enfoque en seguridad y buenas prácticas. Incluye un Bastion Host para acceso SSH seguro, un servidor web en EC2 y una base de datos PostgreSQL en RDS, todo definido como código mediante Terraform.
