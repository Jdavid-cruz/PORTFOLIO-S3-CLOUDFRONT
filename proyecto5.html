Resumen del Proyecto

Este proyecto representa una infraestructura base para entornos tradicionales en AWS. Se crearon redes personalizadas (VPC), subredes públicas y privadas, se configuró un gateway NAT, y se desplegaron instancias EC2 para pruebas de despliegue. Todo fue definido con Terraform como infraestructura como código (IaC).
